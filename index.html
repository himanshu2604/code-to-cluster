// tools & technologies

Tech Stack

🌿
GIT
Version Control
Trunk-based branching with develop → release → master flow. Monthly releases on the 25th.
⚙️
JENKINS
CI/CD Automation
Declarative pipeline triggered via GitHub webhooks. Orchestrates the full build-push-deploy cycle.
🐳
DOCKER
Containerization
Custom image built from Dockerfile. Pushed to Docker Hub. Identical across dev, test, and production.
☸️
KUBERNETES
Orchestration
2-replica deployment for zero downtime. NodePort service exposes the app on port 30008.
🤖
ANSIBLE
Config Management
Playbooks provision all 4 worker nodes with the right software — Jenkins, Docker, Java, and Kubernetes.
🏗️
TERRAFORM
Infrastructure as Code
Provisions AWS EC2 instances, security groups, and networking. Entire infra recreatable in minutes.
